// Функция ТелеграмСервер.ВыполнитьКодСОтветом проверяет значение условия
// 
// Предопределённые условия (Команда, Сообщение, Ответ контекстной клавиатуры и так далее)
// проверяются автоматически.
//
// Пользовательские условия должны возвращать переменную Ответ того типа,
// который задан переменной 'Тип значения' условия
//
// Параметры:
//  1 — Код — строка — код проверки условия, должен присваивать значение переменной Ответ
//  2 — Вход — Структура — входящие данные с ключами, полный список ключей такой же, как в обработках
//
Функция ВыполнитьКодСОтветом(Код, Вход) Экспорт
	
	Ответ = Неопределено;
	Выполнить(Код);
	Возврат Ответ;
	
КонецФункции
