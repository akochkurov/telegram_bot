Получение обновлений

Периодический обмен с серверами Telegram с помощью обработчиков ожидания формы — то есть пока форма "Получение обновлений" открыта, обмен происходит

Обмен через "Получение обновлений" доступен только для ботов со способом связи «Периодические запросы (GetUpdates) через обработку "Получение обновлений"»
